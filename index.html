Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Describe la estructura de una página web mediante etiquetas.

Características principales:

  • Define la estructura y contenido de una página web
  • Utiliza elementos etiquetados como <h1>, <p>, <a>, etc.
  • Se combina con CSS para el estilo y JavaScript para la interactividad
  • Es interpretado por los navegadores web

Estructura Básica HTML

Todos los documentos HTML deben seguir una estructura básica:

  1. <!DOCTYPE html>: Declara el tipo de documento
  2. <html>: Elemento raíz que contiene todo el documento
  3. <head>: Contiene metainformación como título, codificación y enlaces a recursos
  4. <body>: Contiene el contenido visible de la página

Elementos semánticos HTML5:

HTML5 introdujo elementos semánticos que describen mejor su contenido:

  • <header>: Cabecera de la página o sección
  • <nav>: Contiene enlaces de navegación
  • <main>: Contenido principal del documento
  • <section>: Define una sección en un documento
  • <footer>: Pie de página o sección

Ejemplo de Código HTML Básico

Aquí tienes un ejemplo simple de código HTML que puedes usar como plantilla:

<!-- Declaración del tipo de documento -->
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi página web</title>
</head>
<body>
    <header>
        <h1>¡Hola Mundo!</h1>
        <p>Mi primera página web con HTML</p>
    </header>

    <main>
        <section>
            <h2>Bienvenidos</h2>
            <p>Este es un ejemplo de página web básica.</p>
        </section>
    </main>

    <footer>
        <p>© 2023 Mi Página Web</p>
    </footer>
</body>
</html>

Este código incluye todos los elementos esenciales de una página web moderna con HTML5.
